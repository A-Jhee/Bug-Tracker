Tickets

Eyes On The Bug

Use tickets to track a bug from identification to resolution. All team members can submit a ticket to start tracking an issue.

Every ticket features editing, file attachments, and comments. Also features change history so you can make a quick reference when needed.

Dashboard

The Broad Strokes

Every user is greeted to the dashboard upon login.

Interactive graph shows at a glance the number of open tickets and how they fair up to the resolved tickets each day.

Also features a table of all tickets created within the last 3 days. So users are always in the loop.

Projects

Organize Your Tickets

Projects can be used to organize users and tickets under one banner.

Create new projects and track multiple projects, or break up a large project into smaller projects.

User Role Management

Assign Roles On The Fly

Admin can review and change user roles at any time.

For every new user, the role of 'Quality Assurance' is assigned by default.

With a quick select and a click, user roles are immediately effective to grant new role-based permissions.
